📊 Layoffs Data Analysis Project (SQL)

This project focuses on analyzing a real-world layoffs dataset using SQL.

🔧 Key Tasks Performed:
- Data Cleaning
- Removing Duplicates using ROW_NUMBER()
- Handling NULL values
- Standardizing data formats
- Exploratory Data Analysis (EDA)

📈 Insights:
- Identified companies with highest layoffs
- Analyzed layoffs by country and industry
- Found trends over time

🛠 Tools Used:
- MySQL
- SQL Queries
